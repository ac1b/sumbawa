## Active Rules
_(confirmed by data, apply always)_

- **(2026-03-04, confidence: high, evidence: brand-rules.md)** NEVER say "buy land" — use "invest in", "secure", "leasehold opportunity"
- **(2026-03-04, confidence: high, evidence: brand-rules.md)** NEVER guarantee ROI — use "potential", "projections", "historical trends"
- **(2026-03-04, confidence: high, evidence: brand-rules.md)** Always include specific price ($10-18/sqm), location, infrastructure catalyst, Bali comparison
